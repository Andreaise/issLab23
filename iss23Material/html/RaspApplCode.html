python -m pip install -U pip

Software per il Sonar¶

SonarAlone.c

Software per il sonarHCSR04 scritto in C

Sonar.py

Software per il sonarHCSR04 scritto in Python

SonarMqtt.py

Software per il sonarHCSR04 scritto in Python che pubblica dul broker MQTT mqtt.eclipseprojects.io l’evento msg(sonardata,event,sonar,none,distance(D),N).

Si veda mqttPlotQakEvents.py e radargui23.qak possibili receiver.

Software lowlevel per il RadarSystem¶

LedControl.py

Software per controllo del Led in Pipe

python sonar.py | python LedControl.py

radargui23.qak

Modello qak per il radar che percepisce eventi sonardata:distance(D) emessi da SonarMqtt.py

Sistema che accende il Led e aggiorna il radarDisplay:

python sonarMqtt.py | python LedControl.py
